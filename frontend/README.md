## `Install and Run`

Download [ZIP](https://github.com/davitlabadze/mern-todo/archive/refs/heads/master.zip) or Clone repository

```bash
git clone https://github.com/davitlabadze/mern-todo.git
```
#### `Copy .env file`
```bash
cp .env.example .env
```

#### `Install Node modules`

```bash
npm install
```

#### `Run server`

```bash
npm start
```
